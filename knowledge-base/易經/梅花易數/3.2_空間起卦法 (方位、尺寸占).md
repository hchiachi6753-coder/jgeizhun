# 空間起卦法 (方位、尺寸占)

此法是從空間維度入手，觀察事物的方位與尺度來起卦 。


方位占：
o 規則：直接依據文王後天八卦圖所對應的方位取卦。此法通常用於確定外應，或與其他
方法結合使用，例如在動物占中確定下卦。
o 方位對應：震（東）、離（南）、兌（西）、坎（北）、乾（西北）、坤（西南）、艮
（東北）、巽（東南）。



尺寸占：
o 規則 (物體長度大於一丈)：
 上卦 = 丈數 (mod8)。
 下卦 = 尺數 (mod8)。
 動爻 = (丈數 + 尺數) (mod6)。
o 規則 (物體長度小於一丈)：
 上卦 = 尺數 (mod8)。
 下卦 = 寸數 (mod8)。
 動爻 = (尺數 + 寸數) (mod6)。
o 變法：後人應用中，亦有將丈尺（或尺寸）之和再加上起卦時的時辰數，再除以 6 取動
爻，以融入時間因素。
o 注意：測量單位必須使用市制單位（丈、尺、寸）。
