# 萬物類象：八卦的象徵體系

八卦作為宇宙的八種基本原型，其象徵意義可以延伸至大千世界的萬事萬物。正確判斷事物與卦象的
對應關係，是《梅花易數》占卜的第一步 。以下系統性整理八卦在各領域的對應關係：

天時：乾為冰、雹、霰；坤為雲、霧；震為雷；巽為風；坎為雨、雪、露；離為電、霓、虹、
霞；艮為山中雲霧；兌為新月、星辰 。
地理：乾為京都、大城市、高亢之地；坤為田野、鄉里、平地；震為森林、鬧市；巽為草原、
花園、果園；坎為江河、湖泊、沼澤、泉水；離為乾燥之地、窯灶；艮為山脈、丘陵、山路；
兌為澤畔、山崩地裂之地 。
人物：乾為君父、官員、老人；坤為老母、農夫、平民；震為長男、將軍；巽為長女、秀才、
寡婦、隱士；坎為中男、漁夫、盜賊；離為中女、文人、士兵；艮為少男、閒人、山中之人；
兌為少女、歌妓、巫師 。
動物：乾為馬；坤為牛；震為龍；巽為雞；坎為豬、魚類等水生動物；離為雉、鳳凰、龜類等
色彩斑斕的動物；艮為狗、鳥類；兌為羊 。
靜物：乾為金玉、寶珠、鏡子、圓形之物；坤為布帛、絲綿、五穀、方形之物；震為木竹、樂
器；巽為木香、繩子、工巧之器；坎為酒器、水具、帶核之物；離為火、書籍、甲胄；艮為土
石、瓜果；兌為金刃、金屬樂器、破損之物 。
房宅：乾為高樓、廳堂、西北向之居；坤為村居、矮屋、西南向之居；震為山林之宅、東向之
居；巽為寺觀、園林、東南向之居；坎為近水樓台、酒肆、北向之居；離為陽明之宅、南向之
居；艮為山居、近路之宅、東北向之居；兌為近澤之居、斷壁殘垣、西向之居 。
漢字：乾為帶「金」之字；坤為帶「土」之字；震為帶「木」之字；巽為帶「草」之字；坎為
帶「水」之字；離為帶「火」之字；艮為帶「山」或「石」之字；兌為帶「口」之字 。
