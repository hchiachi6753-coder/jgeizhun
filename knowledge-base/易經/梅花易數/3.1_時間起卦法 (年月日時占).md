# 時間起卦法 (年月日時占)

此法是《梅花易數》中最基礎且常用的起卦方式，以占卜當時的時間信息來取數 。




取數規則：
o 年份：取該年農曆地支對應的序數。子為 1，丑為 2，寅為 3，卯為 4，辰為 5，巳為 6，
午為 7，未為 8，申為 9，酉為 10，戌為 11，亥為 12。
o 月份：取農曆月份數。正月為 1，二月為 2，...，臘月為 12。
o 日期：取農曆日期數。初一為 1，初二為 2，...，三十為 30。
o 時辰：取該時辰地支對應的序數。子時為 1，丑時為 2，...，亥時為 12。
o 注意：所有時間信息必須以農曆為準。
起卦公式：
o 上卦：(年支數 + 月數 + 日數) (mod8)。
o 下卦：(年支數 + 月數 + 日數 + 時辰數) (mod8)。
o 動爻：(年支數 + 月數 + 日數 + 時辰數) (mod6)。
