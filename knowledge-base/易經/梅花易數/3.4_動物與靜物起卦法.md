# 動物與靜物起卦法

此法是根據特定動物的屬性或人造靜物的生成時間來起卦 。


動物占：
o 規則：
 上卦 = 根據動物的八卦屬性直接確定（參見表格 1）。
 下卦 = 根據動物來臨的方位或其所處的方位確定。
動爻 = (上卦先天數 + 下卦先天數 + 時辰數) (mod6)。
o 注意：此法通常適用於單個動物，若為一群則不宜使用。
靜物占：
o 規則：此法不適用於山、石、河流等自然靜物，而專指人工製造或種植的靜物（如房
屋、桌椅、樹木等）。因其皆有確定的生成、竣工或種植時間，故直接採用其「生成時
間」，轉化為年月日時占法進行起卦。
o 原則：應用此法需遵循「無疑不占，無故不占」的原則，不可隨意見物起卦。

第四部分：斷卦核心理論：體用學說
體用學說是《梅花易數》的精髓所在，是從卦象推導出吉凶判斷的橋樑與核心算法。它通過確立卦象
中的主體（體）與客體（用），並分析兩者之間的五行生剋關係，來預測事物的發展趨勢與最終結
果 。
