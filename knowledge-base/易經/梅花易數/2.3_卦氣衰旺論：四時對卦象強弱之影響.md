# 卦氣衰旺論：四時對卦象強弱之影響

卦氣衰旺論是《梅花易數》中至關重要的「情境化」調節機制，它防止了占卜淪為脫離現實的純數字
遊戲。一個卦象的吉凶並非一成不變，而是受到其所處「時間背景」的深刻影響 。此理論由漢代易學
家孟喜、京房創立，將八卦的五行屬性與四季氣候相結合，用以判斷卦氣的強弱 。




卦氣旺 當卦的五行屬性與季節的五行屬性相同或得到季節五行的生助時，此卦之氣為「旺」，
其在斷卦中的力量（無論是生助還是剋伐）會被增強。
o 春季（木）：震卦（木）、巽卦（木）為旺。
o 夏季（火）：離卦（火）為旺。
o 秋季（金）：乾卦（金）、兌卦（金）為旺。
o 冬季（水）：坎卦（水）為旺。
o 四季末月（辰、戌、丑、未月，屬土）：坤卦（土）、艮卦（土）為旺 。
卦氣衰 當卦的五行屬性被季節的五行所剋制時，此卦之氣為「衰」，其力量會被削弱。

春季（木剋土）：坤卦（土）、艮卦（土）為衰。
夏季（火剋金）：乾卦（金）、兌卦（金）為衰。
秋季（金剋木）：震卦（木）、巽卦（木）為衰。
冬季（水剋火）：離卦（火）為衰。
四季末月（土剋水）：坎卦（水）為衰 。

例如，一個屬木的體卦（代表自己），在春天（木旺）遇到屬金的用卦（剋我）時，尚有抵抗之力；
若在秋天（金旺木衰）遇到，則凶險倍增。這要求 GPTs 在進行斷卦時，必須將「當前時間」（特別是
農曆季節）作為一個關鍵的輸入參數，以實現對吉凶程度的精準判斷。

第三部分：《梅花易數》起卦方法大全
《梅花易數》以其起卦方式的靈活多樣而著稱，其核心思想是「萬物皆可為數，萬數皆可成卦」。無
論是動還是靜，時間還是方位，大千世界萬事萬物無不可以轉化為《易經》象數 。以下系統性地歸納
和詳解書中所有具体的起卦方法，每種方法都按照「取數規則 -> 求上卦 -> 求下卦 -> 求動爻」的標
準流程進行拆解。
