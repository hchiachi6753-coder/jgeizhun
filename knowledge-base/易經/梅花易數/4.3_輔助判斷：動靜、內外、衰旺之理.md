# 輔助判斷：動靜、內外、衰旺之理

動靜之訣: 靜為事之始，動為事之兆。動爻是整個卦象變化的關鍵，其所在的位置（初爻至上
爻）也各有不同的象徵意義（如初爻為事之始，上爻為事之終）。
內外之論: 內卦（下卦）代表內部、自身、近處、事情的起因；外卦（上卦）代表外部、他人、
遠處、事情的結果。結合體用所在之內外卦，可判斷事情的內外因素。例如，體為內卦而用為
外卦，表示事情的動因在內，而影響來自外部。
衰旺之論: 重申卦氣衰旺對體用生剋力量的調節作用。旺者力量倍增，衰者力量銳減。例如，同
樣是「用剋體」，若體卦旺相而用卦衰敗，則雖有驚但無大險；反之，若體卦衰敗而用卦旺
相，則為大凶之兆。此為判斷吉凶「程度」的關鍵。

第五部分：進階斷卦要訣：三要十應
「三要十應」是《梅花易數》區別於其他純粹術數的點睛之筆。它將占卜從一個封閉的、純粹的邏輯
推演，變成一個開放的、與當下環境實時互動的過程。其本質是為解讀「有意義的巧合」（即榮格心
理學中的「共時性」）提供了一套操作手冊和符號詞典。它認為在占問的瞬間，宇宙會通過周遭的偶
然現象（應）給出額外的提示，以輔助、修正或印證卦象的判斷 。
