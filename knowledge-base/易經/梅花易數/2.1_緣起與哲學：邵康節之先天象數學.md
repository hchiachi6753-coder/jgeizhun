# 緣起與哲學：邵康節之先天象數學

邵雍（1011-1077），字堯夫，諡康節，是北宋著名哲學家，與周敦頤、張載、程頤、程顥並稱「北宋
五子」，為宋明理學的主要奠基人之一 。邵康節精通先天象數之學，其所創立的《梅花易數》，標誌
著中國古代預測學的一次深刻革命——從神秘的、儀式化的占卜（如龜卜、蓍筮）轉向一種更具邏輯
性、可操作性的「數理推演模型」。
其核心貢獻是建立了一套普適的「編碼-解碼」系統。此系統的哲學基礎源於其先天象數學思想，認為
「太極不動，性也，發則神，神則數，數則象，象則器」，即宇宙萬物皆由太極生出，其生成過程遵
循著數、象、器的邏輯序列。因此，萬物皆有其內在的「數」，此數可通過外在的「象」來觀察和推
演。這為「萬物皆可起卦」提供了堅實的哲學依據 。邵康節將一切天時、地理、聲音、人物、顏色、
形狀等世間萬象，通過簡單的數學運算「編碼」為卦象，再通過「體用」等規則對卦象進行「解
碼」，從而得出預測 。這整個流程，極大地削弱了占卜的神秘主義色彩，而增強了其作為一種「宇宙

信息解碼術」的工具理性色彩，這也是它為何能與現代計算機思維（算法、數據庫）如此契合的根本
原因。
