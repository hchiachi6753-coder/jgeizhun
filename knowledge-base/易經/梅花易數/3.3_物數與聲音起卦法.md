# 物數與聲音起卦法

此法是根據可數的物體數量或聲音次數來起卦，應用極為廣泛 。




物數占：
o 規則：
 上卦 = 物體數量 (mod8)。
 下卦 = 時辰數 (mod8)。
 動爻 = (物體數量 + 時辰數) (mod6)。
o 擴展應用：此法可靈活應用於現代生活中的各種數字序列，如電話號碼、車牌號、門牌
號等。例如，可取號碼的某幾位數字之和來求上卦，另幾位之和求下卦。
聲音占：
o 規則：
 上卦 = 聽到的聲音次數 (mod8)。



下卦 = 時辰數 (mod8)。
動爻 = (聲音次數 + 時辰數) (mod6)。
