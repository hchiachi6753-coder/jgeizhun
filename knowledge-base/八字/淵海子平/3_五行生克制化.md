# 五行生克制化



五行相成：
o 金旺得火方成器皿。
o 火旺得水方成相濟。
o 水旺得土方成池沼。
土旺得木方能疏通。
o 木旺得金方成棟梁。
五行過多反受害：
o 土多金埋，火多土焦，木多火熾，水多木漂，金多水濁。
五行生反受害：
o 水多金沉，木盛水縮，火多木焚，土多火埋，金多土變。
五行克反受制：
o 木堅金缺，土重木折，水多土流，火多水熱，金多火熄.
五行衰弱受制：
o 金衰遇火必銷鎔，火弱逢水必熄滅，水弱逢土必淤塞，土衰遇木必傾陷，木弱逢金必砍
折。
