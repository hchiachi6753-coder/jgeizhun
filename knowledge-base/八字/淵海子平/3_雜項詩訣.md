# 雜項詩訣

《刑沖》詩訣：論比肩陽刃與刑沖對六親的影響。
《飄蕩》詩訣：論偏財與離鄉、風流的關係。
《帶疾》詩訣：論五行偏枯與疾病的關係。
《壽元》詩訣：論壽命長短的判斷。
《克妻》詩訣：論財星與陽刃逢沖對妻妾的影響。
《運晦》詩訣：論比肩劫財逢運的兇險。
《長生》詩訣：論長生在命中的吉兆。
《沐浴》詩訣：論沐浴桃花對婚姻、品行的影響。

《冠帶》詩訣：論冠帶對家業的影響。
《臨官》詩訣：論臨官在事業上的吉兆。
《帝旺》詩訣：論帝旺對祖業和名聲的影響。
《衰病死》詩訣：論納音衰病死對吉凶的影響。
《墓庫》詩訣：論墓庫與財富、性情的關係。
《絕胎》詩訣：論絕胎對命運的反本還元。
《胎養》詩訣：論胎養的半凶半吉。
