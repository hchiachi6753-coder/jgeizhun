# 十二地支，中藏何物 (藏干/人元)？

o 子：癸水。
o 丑：己土、辛金、癸水。
o 寅：甲木、丙火、戊土。
o 卯：乙木。
o 辰：戊土、乙木、癸水。
o 巳：丙火、戊土、庚金。
o 午：丁火、己土。
o 未：己土、丁火、乙木。
o

申：庚金、壬水、戊土。
o 酉：辛金。
o 戌：戊土、辛金、丁火。
o 亥：壬水、甲木。
