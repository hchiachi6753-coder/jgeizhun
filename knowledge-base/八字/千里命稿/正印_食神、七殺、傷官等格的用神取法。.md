# 食神、七殺、傷官等格的用神取法。

用神之需要：(一) 有勢有力 (如甲木，當春月)。(二) 有援助 (如甲木，見乙木或癸水之
助)。(三) 在干得氣 (如甲木，支見寅卯)。(四) 在干不見克合 (如甲木，無庚辰己合)。
(五) 在支得生助 (如己火，得甲木資生，丙火幫助)。(六) 在支無刑沖合害 (如己火，不
見亥卯刑)。
