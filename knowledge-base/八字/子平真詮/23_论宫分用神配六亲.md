# 论宫分用神配六亲

人有六亲，配之八字，亦存于命。
其由宫分配之者，则年月日时，自上而下，祖父妻子，亦自上而
下。以地相配，适得其宜，不易之位也。
其由用神配之者，则正印也母，身所自出，取其生我也。若偏财受
我克制，何反为父？偏财者，母之正夫也，正印为母，则偏才为父矣。
正财为妻，受我克制，夫为妻纲，妻则从夫。若官煞则克制乎我，
何以反为子女也？
官煞者，财所生也，财为妻妾，则官煞为子女矣。至于比肩为兄
弟，又理之显然者。
其间有无得力，或吉或凶，则以四柱所存或年月或日时财官伤刃，
系是何物，然后以六亲配之用神。局中作何喜忌，参而配之，可以了然
矣。
