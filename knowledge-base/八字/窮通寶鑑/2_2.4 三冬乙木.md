# 2.4 三冬乙木

總論： 冬季之乙木，花木凋零，寒氣逼人。調候總綱是向陽取暖。丙火是冬乙最核心的用神，有「寒
谷回春」之象。丁火為燈燭之火，力微，不足以解嚴寒之凍。水為大忌，因冬水寒冷，多則冰凍木
根，使其漂浮。
十月乙木 (亥月)



A. 核心環境診斷: 十月建亥，壬水司令，乙木在此處為死地。水冷木凍，乙木不受氣。核心
「病症」是水旺寒凍。
B. 主要調候用神與輔助: 針對「水旺寒凍」，首要藥方是丙火，用以解凍。其次，因水勢過
旺，需戊土為堤壩，制水護丙。 最佳格局是「丙戊兩透，科甲定然」。
AI 判斷規則：
最高級： IF (日主=乙, 月令=亥) AND (丙戊兩透) -> Outcome: 科甲定然
o 次高級： IF (日主=乙, 月令=亥) AND (有丙無戊) -> Outcome: 雖不科甲，亦入儒林

十一月乙木 (子月)


A. 核心環境診斷: 十一月建子，一陽來復，但天氣依然寒凍。子水為癸水之祿，水勢更純更
寒。乙木在此為病地，生機微弱。核心「病症」是極度寒凍。



B. 主要調候用神與輔助: 此月對丙火的需求達到頂點。「喜用丙火解凍，則花木有向陽之
意」。不宜用癸水，因會加重寒氣。
最佳格局是有一二點丙火出干，且無癸水克制，可許科甲。即使丙火藏於地支（如寅、巳），
亦有功名 。

o 癸制丁： 冬月乙木，丁火力弱，若再見癸水克制，則全無用處。
