# 五行之性與生剋制化

《窮通寶鑑》開篇即闡明五行之根本，其論述並非將金、木、水、火、土視為靜態的物質，而是將其
定義為宇宙間循環不息的動態力量，故謂之「行」。此觀點為構建八字分析系統奠定了第一塊基石：
所有元素皆處於永恆的運動與轉化之中。
五行的起源被追溯至天地間的氣候與方位現象：北方陰氣至極而生寒，寒氣凝聚成水；南方陽氣至極
而生熱，熱量升騰為火；東方陽氣發散而生風，風氣催生為木；西方陰氣收斂而生燥，燥氣凝結成
金；中央陰陽交合而生溫，溫潤之氣化育為土 。此源流揭示了五行與自然環境的深刻連結，為後續
「調候」理論（即根據出生月份的氣候調整八字平衡）提供了根本依據。
五行不僅具備物理屬性，更被賦予了形而上的德性：木主仁，其性騰上而無所止；火主禮，其性炎
上；金主義，其性沉下而有所止；水主智，其性潤下；土主信，其性重寬厚博，無所不容 。此德性框
架為八字命理的人格特質分析提供了理論模型。其中，土的中心地位被特別強調，它承載萬物，為其
他四行的生發與歸宿提供了必要的媒介：「金不得土，則無自出；火不得土，則無自歸...故五行皆賴土
也」。
五行間的相互關係，是維持宇宙動態平衡的核心機制。書中明言：「其相生也所以相維，其相克也所
