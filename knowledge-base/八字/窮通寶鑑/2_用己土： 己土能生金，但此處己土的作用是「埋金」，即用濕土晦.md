# 用己土： 己土能生金，但此處己土的作用是「埋金」，即用濕土晦金之光，使其不那么

鋒利。同時己土也能培乙木之根。 最佳格局是「丙透干，又加己出埋金」，主科甲之
命。若僅有己透加丙，亦是上等命格 。
AI 判斷規則：
最高級： IF (日主=乙, 月令=申) AND (丙己齊透) -> Outcome: 科甲上命
o 次級： IF (日主=乙, 月令=申) AND (癸透丙藏，庚少) -> Outcome: 不用己土，可許貢
拔
o 中級： IF (日主=乙, 月令=申) AND (無丙，有癸透) -> Outcome: 刀筆門戶
C. 不良格局與忌神:
o 庚多無救： 若地支庚金過多，癸水又藏，而無丙火己土來救，為平常人物 。

根枯： 秋季乙木最忌根枯，若八字中無水滋潤，根基枯槁，主貧苦到老 。
D. 特殊組合與變化:
o 從化格： 若生於乙巳日、甲申時，或八字組合符合乙庚化金的條件，則可論「從化
格」，反主富貴。化金格以土（印）為用神，忌火來克破。從化格以火為妻，土為
子 。

o 用己土者，以火為妻，土為子 。

八月乙木 (酉月)



A. 核心環境診斷: 八月建酉，辛金司令，乙木處絕地。此時禾稼收割，芝蘭枯萎。乙木被比喻
為丹桂，需分秋分前後論。秋分前，桂花未開，需水滋養；秋分後，桂花已開，喜火向陽。核
心「病症」是金旺木絕，氣候寒涼。



B. 主要調候用神與輔助: 調候需根據節氣細分：
o 白露後，秋分前： 桂蕊未開，端用癸水以滋潤桂萼 。
o 秋分後： 桂花已開，喜向陽，宜用丙火，癸水次之 。
最佳格局是「丙癸兩透」，無論秋分前後，皆主科甲名臣 。

劣級： IF (日主=乙, 月令=酉) AND (四柱不見丙癸) -> Outcome: 下格
C. 不良格局與忌神:
o 支成金局： 若地支巳酉丑合成金局，金氣過旺，必須暗藏丁火制金，否則木被金傷。若
無水火，一生勞碌 。
o 土多： 八月乙木，木已衰弱，若再見戊己土多，耗其元氣，為下格 。
o 強金克木： 書云：「甲乙遇強金，魂歸西土」，「乙木生居酉，莫逢巳酉丑」，皆言秋
木逢旺金，再行金鄉，有性命之憂 。
E. 六親關係指定:
o 用癸者，金妻、水子。
o 用丙者，木妻、火子。
o 用壬者，金妻、水子 。



A. 核心環境診斷: 九月建戌，戊土司令。此時「根枯葉落」，乙木氣息奄奄。戌為燥土，耗水
傷木。核心「病症」是土燥木枯。
B. 主要調候用神與輔助: 針對「土燥木枯」，滋潤是第一要務。故「必賴癸水滋養」。
為確保癸水有源，喜見辛金發水之源。 最佳格局是見癸水，又遇辛金，定主科甲。 AI 判斷規
則：
o

次級： IF (日主=乙, 月令=戌) AND (有癸無辛) -> Outcome: 常人
o 劣級： IF (日主=乙, 月令=戌) AND (有辛無癸) -> Outcome: 貧賤
C. 不良格局與忌神:
o 壬多： 九月土旺，雖喜水潤，但不喜壬水。因壬水為江湖之水，水大土崩，反而難以生
木，故見壬水多者，亦是尋常之輩 。
o 從才格： 若地支戊土多，天干又透，不見比劫，可作「從才格」論。一見比劫，則為富
屋貧人 。
