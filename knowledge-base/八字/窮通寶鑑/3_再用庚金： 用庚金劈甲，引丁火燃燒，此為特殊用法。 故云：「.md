# 再用庚金： 用庚金劈甲，引丁火燃燒，此為特殊用法。 故云：「獨愛丁火，壬癸滋

扶」。
最佳格局是「丁壬癸透，戊己亦透」，此為中和之象，可許一榜。若再有庚金得所，則
科甲定然 。

C. 不良格局與忌神:
o 比肩無制： 若見一二比肩（甲乙），而無庚金制之，木來克土，為平常之人，運氣不佳
時甚至貧無立錐 。

棄命從才： 若柱中多見戊己土，不見比劫印綬，可作「棄命從才格」論，取火為妻，土
為子 。
o 火多傷金： 若一派丙丁火，克傷庚金，又無壬癸水來破火，此為假斯文，若支再成火
局，木成枯朽，乃孤貧下賤之輩 。
D. 特殊組合與變化: 「假傷官，得地逢生，此正合甲乙秋生貴元武之說」。元武指水。此言秋
木若為傷官格（丁火旺），喜見水（元武）來制伏傷官，此為貴格。此時以金為妻，水為子。
E. 六親關係指定: 九月甲木用神複雜，六親判定也隨之變化。
o 用火者，木妻火子。
o 用水制傷官者，金妻水子。
o 從才格者，火妻土子。
o 書中特別指出，九月不取「土妻庚子」，而應取「水妻木子」。此為通用原則，因九月
土燥為病，不宜再以土為妻星。
