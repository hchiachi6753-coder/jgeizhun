# 3.3 三秋丙火

總論： 秋季之丙火，太陽西斜，光輝漸弱。秋季金當令，土亦不弱，金多洩火氣，土多晦火光。此時
的丙火，依然需要壬水來輔映其「暮夜光天」的餘暉。若壬水過多，則需戊土來制水。故秋丙之用，
多在壬、戊之間進行權衡。
七月丙火 (申月)


A. 核心環境診斷: 七月建申，申宮庚金司令，壬水長生。此時「太陽轉西，陽氣衰矣」。丙火
處於病地，氣勢衰弱。同時，申月金旺水相，財殺強盛。核心「病症」是日主衰弱，財殺強
旺。



B. 主要調候用神與輔助: 此時仍用壬水輔映光輝，但因壬水在申宮得長生，氣勢亦強，若壬水
過多，則需戊土出干制之。 最佳格局是壬水與戊土俱透天干，可云科甲。 AI 判斷規則：
o 貴格： IF (日主=丙, 月令=申) AND (壬戊俱透) -> Outcome: 科甲
o 次貴： IF (日主=丙, 月令=申) AND (戊藏支內，壬透) -> Outcome: 不過生員
o 特殊貴格： IF (日主=丙, 月令=申) AND (多壬，一戊出制) -> Outcome: 眾殺猖狂，一
仁可化，主權高位顯
C. 不良格局與忌神:
o 多壬無戊： 壬水七殺過多而無戊土制伏，為平常人 。
o 戊多壬少： 戊土食神過多而壬水七殺過少，亦屬常人 。





D. 特殊組合與變化:
o 棄命從才： 若一派辛金（正財），不見比劫（丙丁），可構成「棄命從才格」，為奇特
之造，亦主富貴。但多依靠親戚或妻家而進身。從才者以水為妻，木為子 。

八月丙火 (酉月)



A. 核心環境診斷: 八月建酉，辛金司令。此時「日近黃昏」，丙火之餘光，存於湖海。丙火處
死地，氣勢衰絕。核心「病症」與七月同，為日主衰絕，財星旺盛。
B. 主要調候用神與輔助: 調候之法，依然是「仍用壬水輔映」。
此月丙火極弱，若柱中多見丙火比肩，得一壬水高透，反為奇格，定主登科及第，富貴雙全。
若壬水藏支，亦主秀才 。
AI 判斷規則：
奇格： IF (日主=丙, 月令=酉) AND (四柱多丙，一壬高透) -> Outcome: 登科及第，富
貴雙全
o 次格： IF (日主=丙, 月令=酉) AND (一壬藏支) -> Outcome: 秀才
C. 不良格局與忌神:
o 戊多困水： 若戊土過多，克制壬水，則為假斯文 。
o 無壬用癸： 若無壬水，不得已用癸水，功名不久 。
o 丙辛合： 若見辛金透干，與丙作合，不能從化，主貧苦到老。若有一丁火制辛，則為人
奸詐。女命若此，長舌淫賤 。
o 從才格： 若地支合成金局（巳酉丑），辛金出干，不見比劫，此為「從才格」，反主富
貴，得妻助。此格用水者，金妻水子；從才者，水妻木子 。
o

九月丙火 (戌月)



A. 核心環境診斷: 九月建戌，戌為燥土，為火之墓庫。此時「火氣愈退」，戊土司令，晦火之
光極甚。核心「病症」是土重晦光，日主入墓。
B. 主要調候用神與輔助: 針對「土重晦光」，必須先用甲木來疏通厚土，劈開墓庫，使丙火得
以重見光明。其次才用壬水來輔映。 故云：「必須先用甲木，次取壬水」。
最佳格局是「甲壬兩透，富貴非凡」。
AI 判斷規則：
最高級： IF (日主=丙, 月令=戌) AND (甲壬兩透) -> Outcome: 富貴非凡
o 次級： IF (日主=丙, 月令=戌) AND (無壬有癸透干) -> Outcome: 異路功名
o 中級： IF (日主=丙, 月令=戌) AND (甲藏壬透，無庚破甲) -> Outcome: 可許秀才
C. 不良格局與忌神:
o 庚戊困水木： 若庚金克甲木，戊土克壬水，用神俱傷，定是庸才 。
o 無甲壬癸： 若甲、壬、癸全無，為下格 。
o 一派火土： 若柱中一派火土，土燥火晦，若非過繼他人，亦主四處漂泊。若再無金水出
干，必為夭折之命 。
o

運入南方火地，反主一貧徹骨 。
