# 2.1 三春乙木

總論： 春季之乙木，被喻為芝蘭蒿草，其性柔弱。與甲木不同，乙木不喜金克，而喜陽光雨露的滋
養。因此，丙火（太陽）與癸水（雨露）是春季乙木的兩大核心用神，不可或缺。書云：「乙木根荄
種得深，只須陽地不宜陰」，陽地即指丙火，陰即指過多的水。丙癸齊透，無克合，主科甲及第。
正月乙木 (寅月)



A. 核心環境診斷: 正月乙木，雖處生旺之地，但同甲木一樣，天氣尚有「余寒」。乙木為花
草，比甲木更畏寒冷。因此，核心「病症」是寒濕。
B. 主要調候用神與輔助: 針對「余寒」，首要藥方是丙火，用以取暖向陽。故云：「必須用
丙，因天氣尤有余寒，非丙不暖」。其次，需要
癸水滋潤根基。 最佳格局是「丙癸兩透，科甲定然」。

AI 判斷規則層次：
o

最高級： IF (日主=乙, 月令=寅) AND (丙癸兩透) -> Outcome: 科甲定然

次高級： IF (日主=乙, 月令=寅) AND (有丙無癸) -> Outcome: 門戶闡揚
o 中級（病態）： IF (日主=乙, 月令=寅) AND (丙多乏癸) -> Outcome: 春旱，獨陽不
長，濁富之人
o 劣級（病態）： IF (日主=乙, 月令=寅) AND (丙少癸多) -> Outcome: 困丙，終為寒士
C. 不良格局與忌神:
o 水多土濕： 若癸水過多，或地支見丑、辰等濕土，水多濕氣重，乙木根易腐爛，為下
格 。
E. 六親關係指定:
o 用丙者，木妻火子。
o 用癸水（且火多）者，金妻水子 。
o

二月乙木 (卯月)



A. 核心環境診斷: 二月建卯，為乙木之臨官祿位，木氣極旺。此時陽氣漸升，木已不畏寒冷。
核心「病症」是木氣過旺，需要洩秀或適當控制。
B. 主要調候用神與輔助: 調候之法，依然是以丙癸為核心。「以丙為君，癸為臣」。丙火用以
洩發旺木之秀氣，構成「木火通明」；癸水用以滋潤，防止木氣因過旺而變得枯燥。
最佳格局是「丙癸兩透，不透庚金，大富大貴」。
AI 判斷規則：
最高級： IF (日主=乙, 月令=卯) AND (丙癸兩透，無庚) -> Outcome: 大富大貴
o 特殊情況： IF (日主=乙, 月令=卯) AND (支成木局，有癸透) -> Outcome: 貴命。若再
得丙洩木氣，為上上之命
C. 不良格局與忌神:
o 水多困丙： 水過多會克傷丙火，用神被制，為下格 。
o 土多化癸： 戊土過多會合去癸水，亦為下格 。
o 埋根之鐵： 乙木為活木，最忌地支藏有庚辛申酉之金，來戕害其根，主木朽損 。
D. 特殊組合與變化:
o 乙庚化金： 若天干透出庚金，地支無辰（辰為水庫，能生木，阻礙化金），則乙庚有作
合化金的可能。若化金成功，則另當別論。若有癸水透出養木，則不以化格論，亦可為
貴。若地支有水庫（如辰），則為假化，平常人 。
o

E. 六親關係指定:
o 用丙者，木妻、火子。
o 用癸者，金妻、水子 。

三月乙木 (辰月)


A. 核心環境診斷: 三月建辰，陽氣愈發熾烈，乙木之氣漸衰。辰中戊土司令，耗洩乙木。核心
「病症」是陽燥土厚，乙木漸弱。



B. 主要調候用神與輔助: 此時對水的需求超過了對火的需求。故「先癸後丙」。首先需要癸水
來滋潤乙木，並潤澤燥土。其次才需要丙火來照耀，洩其秀氣。
最佳格局是「癸丙兩透，不見己庚，玉堂之客」。己土會濁水，庚金會合乙，皆為破格之神。
AI 判斷規則：
最高級： IF (日主=乙, 月令=辰) AND (癸丙兩透，無己庚) -> Outcome: 貴為朝臣
o 次級： IF (日主=乙, 月令=辰) AND (一乙逢庚，無己) -> Outcome: 小富貴，不顯達
o 特殊情況： IF (日主=乙, 月令=辰) AND (支合水局，丙戊高透) -> Outcome: 亦主科甲
C. 不良格局與忌神:
o 水多見土： 水多而見己土，己為濕泥，水土混雜，雖有才華也難以考取功名。若見戊
土，則戊為高亢之土，能制水，或可發於異途 。
o 庚己混雜： 若庚金、己土齊出，而丙癸俱全，格局混亂，為下格 。
o

水多無丙戊： 若支合成水局，而無丙火溫暖、戊土制水，為離鄉背井之命 。
o 壬水泛濫： 若見一派壬水，不特貧賤，而且夭折。需有戊己土制之，方能有壽，但終究
是技術之人 。
D. 特殊組合與變化:
o 二庚爭合： 若生於庚辰時，月令又為庚辰，名「二庚爭合」，為貧賤之輩。若年干有丁
火破庚，可作從化（乙庚化金）論，或有武職之權 。
E. 六親關係指定:
o 用癸者，金妻水子。
o 癸多用丙者，木妻火子 。
