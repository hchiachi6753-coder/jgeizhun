# 3.1 三春丙火

總論： 春季之丙火，太陽之光輝隨著陽氣回升而日益強盛，有「侮雪欺霜」之威 。此時的丙火，最
需要江河湖海之水（壬水）來映照其光輝，形成水火既濟的局面。故春丙以
壬水為最核心的用神，名曰「天和地潤，既濟功成」。正月用壬，需庚辛金來生助；二月專用壬水；
三月因土重晦光，需取甲木來疏土，輔助丙火。春季的丙火，不喜癸水，癸水為雲霧，會遮蔽太陽光
輝，形成「不晴不雨之天」，主不顯不達 。
正月丙火 (寅月)



A. 核心環境診斷: 正月建寅，寅為丙火之長生之地，內藏甲木，能生丙火。此時「三陽開泰，
火氣漸炎」。丙火得祿得生，氣勢漸旺。此時的丙火並非身弱，而是需要水來調候，以成水火
既濟。



B. 主要調候用神與輔助: 針對「火氣漸炎」，首要藥方是壬水，用以輔映太陽之光。其次，春
季水氣不足，需庚金為源，生助壬水。 最佳格局是「壬庚兩透，科甲定然」。
AI 判斷規則層次：
最高級： IF (日主=丙, 月令=寅) AND (壬庚兩透) -> Outcome: 科甲定然
o 次高級： IF (日主=丙, 月令=寅) AND (壬透庚藏) -> Outcome: 異途顯達
o 中級： IF (日主=丙, 月令=寅) AND (一庚高透，支藏丙火) -> Outcome: 納粟奏名，為
人慷慨英雄
C. 不良格局與忌神:
o 庚辛混雜： 若庚辛金混雜，為財星混雜，格局不清，為常人 。
o



貪合： 若年、時為辛，與日主丙火作合，名為「貪合」，主貪戀酒色 。
o 殺重身輕： 若丙火少而壬水多，又無戊土來制壬水，名「殺重身輕」。主其人陰險狡
詐，為是非之徒。若有一戊制壬，反成富貴，但需有比肩（丙丁）相助 。
o 戊土晦光： 正月丙火，最忌戊土出干。戊為燥土，能晦丙火之光，使太陽蒙塵。若一片
戊土，又無甲木出干疏通，終非大器，恐孤貧 。
o 支成火局： 若地支寅午戌合成火局，火勢過旺。此時急需壬水為救。若無壬水，可用癸
水。若壬癸俱無，則只能取戊土洩火氣，但只屬平人。若構成「炎上格」，需行東南木
火運，否則反致孤貧 。
E. 六親關係指定:
o 用壬者，金妻水子。
o 用庚者，土妻金子 。
o

二月丙火 (卯月)



A. 核心環境診斷: 二月建卯，卯為乙木，是丙火的沐浴之地。此時「陽氣舒升」，木旺火相，
丙火氣勢比正月更強。卯為正印，印綬當令，丙火有力。
B. 主要調候用神與輔助: 調候之法與正月相同，依然是「端用壬水」。
最佳格局是壬水透干，不見丁火來合壬，再有庚辛金來生壬水，且壬水有根（地支見亥、子、
申等），定主科甲 。
AI 判斷規則：
貴格： IF (日主=丙, 月令=卯) AND (壬透有根，有金生，無丁合) -> Outcome: 定主科
甲
o 次格： IF (日主=丙, 月令=卯) AND (無壬水，己土姑用) -> Outcome: 主有才學，衣食
充足，但難成名
C. 不良格局與忌神:
o 水多無制： 若一派壬水，水勢過旺，需有一戊制之。雖不能走科甲正途，亦有官職恩
庇。若無戊土，水勢奔流，為下賤之命 。
o 土多： 若一派戊土，亦需用壬水。此時運喜行木運（制土），忌行土運、火運 。
o

D. 特殊組合與變化:
o 從化格： 丙子日辛卯時，丙辛相合，有化水之象。但生於卯月，木旺洩水，不逢其時，
且貪財（辛）壞印（卯），難承祖業。若有丁火破辛金，使丙火不被合，壬水得用，亦
主富貴 。
o 爭合： 若月、時皆為辛卯，日主為丙子，名為「爭合」。若年干無丁火制辛，主其人沉
溺酒色。年透丁火，反吉 。
o 支成木局： 若地支亥卯未合成木局，印綬過旺。主其人可能因奸巧而得財，或因酒色而
得名 。
E. 六親關係指定:
o 凡用壬者，金妻水子 。

三月丙火 (辰月)



A. 核心環境診斷: 三月建辰，辰為濕土，內藏戊土、乙木、癸水。此時「氣漸炎升」，但戊土
司令，土能晦火，洩丙火之氣。核心「病症」是土重晦光。
B. 主要調候用神與輔助: 針對「土重晦光」，調候需雙管齊下。
