# 再用庚金： 庚金作為劈甲之用。 故云：「丁火為先，次用丙火，庚金再次」。

最佳格局是「一丁一庚，科甲定顯」。若丙庚兩透，則富大貴小。
AI 判斷規則：
貴格： IF (日主=甲, 月令=酉) AND (丁庚齊備) -> Outcome: 科甲
o 富格： IF (日主=甲, 月令=酉) AND (丙庚兩透) -> Outcome: 富大貴小
o 富貴雙全： IF (日主=甲, 月令=酉) AND (丙透無癸) -> Outcome: 富貴雙全
C. 不良格局與忌神:
o 癸水出干： 癸水會傷丁火，破壞格局，故「癸水一透，科甲不全」。
o 無丙無丁： 若丙丁火全無，金旺無制，木被金傷，為僧道之命 。
o 支成金局： 若地支巳酉丑合成金局，干又透庚金，為「木被金傷」，必主殘疾。若有丙
丁破金，可減輕，但晚年仍有暗疾 。
