## 🔧 資料格式

### JSON 結構（RAG 使用）

`rag_chunks.json` 包含 1,301 個分塊，每塊約 1,000 字：

```json
{
  "id": "子平真詮_001_chunk_001",
  "text": "天地之間，一氣而己...",
  "source": "子平真詮",
  "chapter": "第1章",
  "title": "論十干十二支",
  "category": "八字",
  "keywords": ["天干", "地支", "陰陽", "五行"]
}
```

### Markdown 結構（人類閱讀）

每個章節獨立存檔，包含 YAML frontmatter：

```yaml
---
source: 子平真詮
chapter: 第1章
title: 論十干十二支
category: 八字
author: 沈孝瞻
dynasty: 清
keywords: ["天干", "地支", "陰陽"]
---
```
