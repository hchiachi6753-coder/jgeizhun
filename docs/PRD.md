## 1. 產品概述

### 1.1 產品名稱
**J給準** (jgeizhun)

### 1.2 產品定位
結合古籍智慧與 AI 技術的紫微斗數命理平台，提供專業級命盤排列與智慧解讀服務。

### 1.3 目標用戶
- 對命理有興趣的一般大眾
- 想了解自己人生走向的人
- 命理師（專業工具）
- 對傳統文化有興趣的年輕族群

### 1.4 核心價值主張
- **準確**：命盤計算與科技紫微網一致
- **專業**：引用古籍原文作為解讀依據
- **現代**：AI 智慧解讀，淺顯易懂
- **賦能**：「命盤是統計，不是限制」

## 3. 設計原則

### 3.1 視覺風格
- **新中式 × 神秘仙氣**
- 主色調：深紫色 + 金色
- 背景效果：紫微星、流星、星雲
- 字體：大字、圓角、舒適比例

### 3.2 用戶體驗
- 視覺優先（80% 用戶不擅長閱讀大量文字）
- 分層展示：摘要 → 卡片 → 展開詳情
- 色彩編碼：吉星綠色、凶星紅色、中性白色
- 100% 行動裝置友善

### 3.3 解讀風格
- 賦能而非宿命論
- 先講重點再展開
- 有根據（引用古籍）
- 不用對話式解讀（太慢）

## 4. 技術架構

### 4.1 前端
- Framework: Next.js 16
- Styling: Tailwind CSS
- 部署: Vercel

### 4.2 後端
- 命盤計算: TypeScript（純前端計算）
- AI 解讀: Claude API
- 知識庫: RAG（待整合）

### 4.3 資料來源
- 農曆轉換: lunar-javascript
- 古籍知識庫: 12 本經典著作
