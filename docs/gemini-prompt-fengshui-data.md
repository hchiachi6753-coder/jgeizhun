## 請產出以下內容

### 1. 擴充 byRoom 建議（每星 x 9種房間）

為以下房間類型提供專屬建議：
- 臥室（主臥）
- 次臥（客房、長輩房）
- 客廳
- 書房
- 廚房
- 小孩房
- 浴室/廁所
- 陽台
- 玄關

**要求：**
- 每個建議 20-40 字
- 要具體可執行（例：「床頭放黃水晶」而非「擺放水晶」）
- 吉位：如何增強運勢
- 凶位：如何化解煞氣
- 考慮房間特性（廚房有火、浴室有水等）

### 2. 擴充化解/增強物品（每星 8-10 個）

**要求：**
- 物品要容易購買（淘寶、IKEA、花市能買到）
- 包含價格區間（平價/中價/高價各幾個）
- 說明為什麼這個物品有效（五行原理）

### 3. 不同身份的專屬建議

為以下身份提供建議：
- forStudent: 學生/考生
- forWorker: 上班族
- forBusiness: 創業者/老闆
- forElderly: 長輩/退休族
- forPregnant: 孕婦
- forSingle: 單身者
- forNewlywed: 新婚夫妻
