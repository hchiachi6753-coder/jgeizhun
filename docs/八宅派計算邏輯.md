# 八宅派計算邏輯

## 一、命卦計算公式

### 立春注意
- 以「立春」（約 2/4）為歲次更替
- 2/4 前出生者，用前一年計算

### 計算公式

**1900-1999 年出生：**
- 男命：`(100 - 出生年後兩位) % 9`
- 女命：`(出生年後兩位 - 4) % 9`

**2000-2099 年出生：**
- 男命：`(100 - 出生年後兩位 + 8) % 9`
- 女命：`(出生年後兩位 + 6) % 9`

### 特殊處理
- 餘數 = 0 → 視為 9（離卦）
- 餘數 = 5 → 男歸坤(2)，女歸艮(8)

| 餘數 | 命卦 | 五行 | 類別 |
|-----|------|-----|------|
| 1 | 坎 | 水 | 東四命 |
| 2 | 坤 | 土 | 西四命 |
| 3 | 震 | 木 | 東四命 |
| 4 | 巽 | 木 | 東四命 |
| 5 | 男→坤/女→艮 | 土 | 西四命 |
| 6 | 乾 | 金 | 西四命 |
| 7 | 兌 | 金 | 西四命 |
| 8 | 艮 | 土 | 西四命 |
| 9 | 離 | 火 | 東四命 |

## 四、應用原則

### 宅命相合
- 東四命 → 宜住東四宅
- 西四命 → 宜住西四宅
- 若不合，調整床頭/書桌朝向補償

### 空間配置建議
- **吉位**（生氣/天醫/延年）：大門、主臥、客廳、財位
- **凶位**（絕命/五鬼/六煞/禍害）：廁所、廚房、儲藏室
