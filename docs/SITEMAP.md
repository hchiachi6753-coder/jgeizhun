## 📍 頁面架構

```
🏠 首頁 (/)
│
├── 🟠 八字命理 (/bazi)
│   ├── 📝 輸入生辰 (/bazi)
│   │   └── [開始解命] → 結果頁
│   │
│   └── 📊 結果頁 (/bazi/result)
│       ├── 八字四柱顯示
│       ├── [全盤解密] → AI 解析
│       ├── [追問命盤] → 追問面板
│       ├── [列印報告] → 瀏覽器列印
│       └── [重新排盤] → 輸入頁
│
├── 🟣 紫微斗數 (/ziwei)
│   ├── 📝 輸入生辰 (/ziwei)
│   │   └── [開始排盤] → 結果頁
│   │
│   └── 📊 結果頁 (/ziwei/result)
│       ├── 紫微命盤顯示 (12宮格)
│       ├── [全盤解密] → AI 解析
│       ├── [追問命盤] → 追問面板
│       ├── [列印報告] → 瀏覽器列印
│       └── [重新排盤] → 輸入頁
│
├── 🟣 綜合解命 (/comprehensive)
│   ├── 📝 輸入生辰 (/comprehensive)
│   │   └── [開始解命] → 結果頁
│   │
│   └── 📊 結果頁 (/comprehensive/result)
│       ├── 八字四柱 + 紫微命盤
│       ├── [全盤解密] → AI 綜合解析
│       ├── [追問命盤] → 追問面板
│       ├── [列印報告] → 瀏覽器列印
│       └── [重新排盤] → 輸入頁
│
└── 🔵 易經占卜 (/yijing)
    ├── 📝 輸入問題 (/yijing)
    │   └── [開始搖卦] → 搖卦頁
    │
    ├── 🎲 搖卦動畫 (/yijing/shake)
    │   └── [搖卦 6 次] → 結果頁
    │
    └── 📊 結果頁 (/yijing/result)
        ├── 本卦/變卦顯示
        ├── [AI 解卦] → AI 解析
        ├── [追問卦象] → 追問面板
        ├── [列印報告] → 瀏覽器列印
        └── [再問一卦] → 輸入問題頁
```

## ⚠️ 功能限制

### 追問功能
| 項目 | 限制 |
|------|------|
| 每張命盤/卦象 | **2 題 / 24 小時** |
| 追蹤方式 | localStorage（按命盤 ID） |
| 命盤 ID 格式 | `年-月-日-時辰-性別` |
| 開發模式 | `?dev=1` 無限追問 |
| 目前狀態 | 🔓 暫時關閉限制（測試中） |

### AI 模型
| 項目 | 設定 |
|------|------|
| 模型 | Gemini Pro 2.5 |
| 專案 | J1-Assistant (Google Cloud) |
| 費用 | 按使用量計費 |

## 🔄 使用者流程 (User Flow)

### 八字/紫微/綜合
```
輸入生辰 → 結果頁 → [全盤解密] → AI 解析
                         ↓
                    [追問命盤] → 選擇分類 → 選擇問題 → AI 回答
                         ↓                              ↓
                    (最多 2 題)                    [繼續追問]
                         ↓                              ↓
                    [列印報告] ← ← ← ← ← ← ← ← ← ← ← ←
```

### 易經
```
輸入問題 → 搖卦動畫 (6次) → 結果頁 → [AI 解卦] → AI 解析
                                         ↓
                                    [追問卦象] → 選擇分類 → 選擇問題 → AI 回答
                                         ↓
                                    [再問一卦] → 回到輸入問題頁（新問題）
```
