# J給準 🔮

> AI 命理解盤平台 - 紫微斗數 × 八字命理

融合千年古籍智慧，以 AI 科技為您解讀命盤。

## ✨ 功能

### 紫微斗數
- 十二宮格命盤顯示
- 14 顆主星 + 6 輔星 + 6 煞星
- 50+ 顆雜曜（博士十二星、長生十二星等）
- 四化飛星
- 大限、流年、流月時間軸

### 八字命理
- 四柱八字排盤
- 十神分析
- 地支藏干
- 大運流程

## 🛠️ 技術棧

- **框架**：Next.js 16 (App Router)
- **語言**：TypeScript
- **樣式**：Tailwind CSS
- **曆法**：lunar-javascript
- **AI**：Claude API (準備中)
- **部署**：Vercel

```
src/
├── app/

│   │   ├── page.tsx      # 輸入頁
│   │   └── result/       # 結果頁
│   └── bazi/             # 八字命理
│       ├── page.tsx      # 輸入頁
│       └── result/       # 結果頁
├── components/
│   ├── ZiweiChart.tsx    # 十二宮格命盤
│   └── JourneyMap.tsx    # 人生旅程圖
└── lib/
    ├── ziwei/            # 紫微計算引擎
    │   ├── index.ts
    │   ├── stars.ts      # 主星安置
    │   ├── minorStars.ts # 雜曜安置

    │   ├── gong.ts       # 宮位
    │   └── sihua.ts      # 四化
    └── bazi/             # 八字計算引擎
```
