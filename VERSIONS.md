# 好運大師 - 版本記錄

## v1.0.0 (2026-02-21) ✅ 目前版本

### 🎴 八字命理
- 四柱排盤（年、月、日、時柱）
- 十神分析
- 五行強弱判斷
- AI 深度解讀（性格、事業、感情、健康）
- 大運流年分析

### ⭐ 紫微斗數
- 十二宮排盤
- 主星、輔星完整安星
- 四化飛星
- AI 深度解讀
- 大限、流年運勢

### ☯️ 綜合解命
- 八字 + 紫微 雙系統合參
- 交叉驗證分析
- 更全面的命理解讀

### ☰ 易經占卜
- 線上搖卦（模擬銅錢）
- 六爻排盤
- 變卦分析
- AI 解讀卦象

### 🧭 居家風水
- AR 羅盤功能
- 空間導覽
- 方位吉凶分析

### 💬 追問功能
- 針對命盤追問
- 6 大分類（事業、感情、財運、健康、學業、決策）
- **限制：每張命盤每 24 小時限問 2 題**

### 🪷 五行能量分布圖
- 蓮花造型視覺化
- 顯示五行強弱
- 缺失元素提示

### 📈 流年運勢曲線
- 10 年運勢走勢圖
- 視覺化呈現高低起伏

### 🎨 設定檔架構
- `site.ts` 集中管理品牌設定
- 改一個檔案即可換品牌

## 📋 技術規格

| 項目 | 規格 |
|------|------|
| 框架 | Next.js 16 + React 19 |
| 語言 | TypeScript |
| 樣式 | Tailwind CSS |
| AI 模型 | Claude Sonnet 4 (`claude-sonnet-4-20250514`) |
