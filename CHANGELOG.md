## 核心算法發現

### 陰陽宮規則

> **陰宮緊密排，陽宮要跳格**

- 紫微星系在陰宮：偏移 0, -1, -3, -4, -5, -8
- 紫微星系在陽宮：偏移 0, -1, -2, -3, -4, -8
- 天府星系在陰宮：偏移 0, +1, +2, +3, +4, +5, +6, +10
- 天府星系在陽宮：偏移 0, +1, +2, +4, +5, +6, +7, +10
